Bento-Glass Component System

Bento Card

Rounded-xl, glassmorphism background, subtle border glow on hover/focus. 12px grid alignment.

Base UnitGlassmorphism12px Grid

Color Pill

Circular (36px) with shape icon + color ring. Pressed state scales to 0.92. Long-press = color info tooltip.

InputHapticTooltip

Viaduct Counter

Icon + count badge. Pulse animation when < 2 remaining. Gradient fill based on availability.

Motion & Interaction Spec

Easing Curves

Standard: cubic-bezier(0.2, 0, 0, 1) — 200ms. Spring: stiffness 300, damping 30 — for pop-ins. Sharp: cubic-bezier(0.4, 0, 1, 1) — 150ms for micro-interactions.

MotionTokens

Gesture Responses

Tap: 80ms scale to 0.95 + haptic light. Drag: follow finger, path preview line. Release: snap to grid, 120ms settle. Long-press: 400ms → tooltip + haptic medium.

TouchHaptic

Screen Transitions

Push (horizontal) for level flow. Modal fade + scale (0.96→1) for crisis/victory. Cross-fade (160ms) for settings/theme change. No transitions during simulation.
